# AWS CloudWatch Health Dashboard

> A lightweight CLI tool that queries CloudWatch metrics across multiple AWS services and outputs a structured, colour-coded health summary.

## The Problem

When responding to an incident an engineer will need to look at the health status of multiple services simultaneously, doing this manually is slow and inefficient.

This tool addresses this issue by pulling the relevant metrics into a single output.

---
## Status

🚧 In active development. See planned features below.

## Planned Features

- Query multiple CloudWatch metrics via a single YAML config
- Threshold-based OK / WARNING / CRITICAL classification
- Colour-coded terminal output
- JSON report export
- LocalStack support for local development without AWS credentials
